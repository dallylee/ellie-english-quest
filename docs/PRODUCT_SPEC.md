# Product Spec

## Product name

Eli’s English Quest

## Product type

A browser-based educational game for one child learner.

## User

Eli:
- age 10
- Croatian L1
- A1 English
- likes singing, hip hop, London, Bjelovar, Maths, Music, Informatics
- needs warmth, confidence, and safe guessing

## Problem

A conversational Custom GPT tutor was not enjoyable or reliable enough. Eli needs something more game-like, structured, and rewarding.

## Solution

A mobile-friendly web game with levels, stars, and short English challenges.

## Primary learning objectives

- Recognise and use everyday A1 vocabulary
- Build confidence answering simple English questions
- Practise word meanings through repetition
- Practise sentence frames without pressure
- Encourage spoken English when browser voice features work

## Design principles

- Game first, lesson second
- Tiny tasks
- Immediate reward
- No shame
- Low reading burden
- Clear visual navigation
- Mobile-first

## MVP scope

- 8 worlds
- 5 modes per world
- local progress
- stars and trophy room
- optional TTS/STT
- deploy as static site

## Out of scope for MVP

- Backend accounts
- Cloud database
- Paid speech APIs
- AI-generated dynamic questions
- Multiplayer
- Teacher dashboard
